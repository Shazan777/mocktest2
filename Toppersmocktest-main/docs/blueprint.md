# **App Name**: PrepAI

## Core Features:

- Class and Board Selection: Allow students to select their class (8, 9, 10, 12) and board (CBSE, UP Board) dynamically.
- Syllabus-based Chapter Display: Dynamically show subject and chapter lists based on class and board selections, following the official NCERT syllabus.
- AI-Generated MCQ Test: Generate 20 unique MCQ questions using AI, following a fixed difficulty pattern (5 easy, 10 medium, 5 hard).
- Timed Mock Test: Implement a 25-minute timer to simulate a real exam environment, with automatic test submission upon timer completion.
- Progress Indicator: Display a progress indicator during the mock test.
- Detailed Performance Report: Provide a comprehensive performance report, including total questions, correct, wrong, skipped, score, accuracy, time taken, and difficulty-wise performance.
- AI-Generated Feedback: Offer AI-generated motivational feedback and learning advice to encourage students based on their performance.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to represent intellect and focus.
- Background color: Very light indigo (#F0E6FF), subtly tinted and highly desaturated to avoid distraction.
- Accent color: Violet (#8F00FF) to add a pop of color without overwhelming the senses.
- Body and headline font: 'Inter', a sans-serif font for readability.
- Use simple, clear icons to represent subjects and functions.
- Clean and intuitive layout with a clear visual hierarchy for easy navigation.
- Subtle animations on transitions and interactions to enhance user experience.
